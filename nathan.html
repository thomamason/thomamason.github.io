a really goofy guide to your softmodded memory card



    Key points:

  • ALWAYS keep this mem card in the second slot (so you don't corrupt your memory card for your saved games!)
  • by default, this is set up to work with a usb harddrive. Use an actual harddrive (not a thumb drive) with a sata to usb cable
  • this guide aims to get you playing games via usb harddrive on your ps2. There's loads more you can do with the software here, but we can talk about that another time. okay lets get into it!

    usage instructions:

  • to start, take your usb harddrive and plug it into a usb port on the front of your ps2. This usb harddrive should be formatted as either FAT32 or ext4
  • once the system is booted, the drive will be populated with a file structure. The only really important folder is called 'DVD'
  • this DVD folder is where you place your ps2 game .iso files. getting these from the web is ILLEGAL so don't do that ;)
  • once you've added games to the DVD folder via your pc, plug the drive in, boot the console up and then navigate to games
  • the program that loads these games is called OPL (open playstation loader) and by default it will search for the hard drive and the games in the .iso folder
  • click to boot the game! EZ

  • UNLESS......

  • not all games will load. Damn near 99%, but not all
  • here's a compatibility list: ps2-home compatibility list

    gamepad emulation

  • i have enabled gamepad emulation for the 1st player by default. you can plug in a DS3 or DS4 game pad into the other usb slot and it should work right out of the box
  • some games really don't like pad ehulation, so if a game won't load, go to options in OPL (triangle button, i think) and disable pademu
